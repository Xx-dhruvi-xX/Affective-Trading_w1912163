## Emotion Dataset
This project uses the FER+ dataset for facial emotion recognition. FER+ (Barsoum et al., 2016) provides improved emotion labels for the FER2013 facial expression dataset introduced by Goodfellow et al. (2013).
- **Images:** FER2013 images are obtained vai Kaggle
- **Labels:** FER+ label files are combined with FER2013 images to form the final training dataset.
- **Privacy:** Only local processing is performed. No images are transmitted or stored during application runtime.
> Note: Dataset files are not committed to Git due to size and licensing restrictions.
